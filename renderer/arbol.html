Árbol de Backtracking

?

Los colores del árbol te ayudarán a identificar el estado de cada nodo durante la ejecución del algoritmo de Backtracking.

El color azul representa nodos normales aún no explorados.

El amarillo indica el nodo que el algoritmo está explorando actualmente.

El verde representa una solución válida que cumple la expresión lógica.

El rojo indica un camino inválido que fue descartado.

El naranja representa el retroceso del algoritmo para probar nuevas posibilidades.
